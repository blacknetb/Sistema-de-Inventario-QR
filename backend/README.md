# Sistema de Inventario con Códigos QR

Sistema completo de gestión de inventario con generación y escaneo de códigos QR.

## Características

- ✅ Gestión completa de productos y categorías
- ✅ Control de inventario con movimientos de entrada/salida
- ✅ Generación de códigos QR para productos
- ✅ Escaneo de códigos QR para consulta de información
- ✅ Reportes en PDF y exportación de datos
- ✅ Alertas de stock bajo
- ✅ Autenticación JWT con roles (admin/user)
- ✅ API RESTful documentada
- ✅ Base de datos MySQL

## Requisitos Previos

- Node.js 14.x o superior
- MySQL 5.7 o superior
- npm o yarn
